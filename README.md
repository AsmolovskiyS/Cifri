# Цифры

![](https://raw.githubusercontent.com/AsmolovskiyS/Cifri/master/Screenshot/Scrn_01.png)

## Описание
Игра написана на Delphi 7.  

## Описание игры
Цель игры - набрать как можно больше очков вычеркивая как можно больше цифр (в идеале вычеркнуть все цифры).  
Необходимо складывать пары одинаковых цифр (*например: 2-2, 5-5, 9-9*) или тех, которые в сумму составляют десять (*например: 1-9, 2-8, 3-7 и т.д.).  
Пары цифр должны располагаться рядом (то есть, между ними могут быть пустые клетки, но не должно быть других цифр). Пары цифр можно собирать по вертикали и горизонтали. По горизонтали пары цифр можно собирать с переходом на следующие ряды.  
Для удаления пары цифр последовательно нажмите на них.  
Если вы длительное время не можете найти пару, то одна из цифр, у которой имеется пара, будет подсвечена.
Если на поле не останется пар, то оставшиеся цифры будут разложены снова со смещением.  

В игре имеется два режима:
+ **Цифры по порядку** (классический режим) - цифры расставляются по порядку.
+ **Цифры в разброс** - цифры расставляются в случайном порядке.  

При запуске новой игры можно указать желаемые размеры поля.
Для каждого размера поля можно установить свой рекорд по набранным очкам. Если у поля уже имеется рекорд, он будет отображаться в правом нижнем углу окна до начала игры.

## Управление
Все управление осуществляется мышкой.
